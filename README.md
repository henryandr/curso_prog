# Curso de Programación

Este repositorio reúne material de estudio y trabajo práctico del curso de programación.

## Objetivo

Centralizar los contenidos de cada unidad para facilitar el aprendizaje, la práctica y la preparación de evaluaciones.

## Contenido del repositorio

- `unidad2/`
 	- `algoritmos.md`: apuntes y ejercicios sobre algoritmos.
 	- `representacion_datos.md`: conceptos de representación de datos.
 	- `taller.md`: actividades prácticas de la unidad.
 	- `examenU2.aux`: archivo auxiliar generado por compilación LaTeX.

## Nota

Los archivos auxiliares de compilación (por ejemplo, de LaTeX) se ignoran mediante `.gitignore` para mantener el repositorio limpio.
